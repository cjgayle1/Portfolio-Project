Absurd Trolley Problems 2 BETA

This game is meant to be an extension of the ABSURD TROLLEY PROBLEMS game made on neal.fun's site. I had a lot of fun playing his game and wanted to create some of my own creative trolley problems. All credit to him for coming up with this idea. My version is not nearly as polished as his so I would highly encourage you to play his here.

Assume that you have the ultimate knowledge in each situation and whatever the scenario says will 100% become true exactly the way it is stated. All drawings were done by me on an iPad using only my finger so get ready for some terrible art.

Warning: This game contains graphic and potentially offensive scenarios that may not be suitable for everyone. These situations are not aimed to attack any belief system, they are simply to question the underlying logic and make the player think. This game was made to be fun and thought-provoking. Please do not take it too seriously. HAVE FUN!
