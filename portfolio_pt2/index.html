AI Recipe Finder

Put in ingredients, get out recipes

Inspired by the challenge of deciding what to cook with on-hand ingredients, I developed a program that suggests recipes based on what users have available. It uses a multi-step process to normalize ingredient data including lemmatization with NLTK and matches recipes by comparing TFIDF-scored vectors and cosine similarities of recipe ingredients against that of the user ingredients. The program will then provide a set number of results that match the users inputs. A user-friendly UI makes it easy for anyone to navigate and discover new meal possibilities.
